# 📊 Central de Scripts SQL - P3/2ª RPM

Este repositório centraliza todos os scripts e consultas SQL utilizados para extração, tratamento e análise de dados da 2ª Região de Polícia Militar.

---

## 📁 Estrutura do Repositório

| Pasta | Descrição |
| :--- | :--- |
| **Apoio Unidades** | Consultas específicas para suporte às unidades operacionais da região. |
| **Demandas Esporádicas** | Scripts criados para atender solicitações pontuais de última hora. |
| **GDO 2024 / 2025** | Consultas voltadas para os indicadores de Gestão de Desempenho Operacional. |
| **Indicadores Táticos** | Scripts de monitoramento de metas e produtividade tática. |
| **Painel Preditivo** | Consultas que construimos para o estudo do painel preditivo. |
| **Violência Doméstica** | Extração de dados específicos para o monitoramento de crimes contra a mulher. |
| **PPAG** | Scripts dos painéis de PPAG do CGA. |
| **Prêmio Tático Móvel** | Scripts utilizados para o dashboard de produtividade do Tático Móvel. |

---

## 🚀 Como utilizar
1. Navegue até a pasta desejada.
2. Abra o arquivo `.sql`.
3. Copie o código e execute no seu gestor de banco de dados (Oracle, SQL Server, etc.).

## 🛠️ Manutenção
Para manter este backup atualizado, novos scripts devem ser submetidos via **Upload** diretamente nas pastas temáticas para garantir a organização.
